# Supply Chain Business Case Study

## Overview

This project demonstrates an end-to-end data analytics workflow using **Python, SQL, and Power BI** to analyze a supply chain dataset and uncover actionable business insights.

The project begins with **data cleaning and preprocessing in Python**, where unnecessary columns were removed, data quality issues were addressed, and the dataset was prepared for analysis. The cleaned data was then imported into **MySQL Workbench**, where a **Star Schema** was designed by creating dimension and fact tables to optimize analytical queries.

Using the star schema, I solved **20 real-world business problems** with SQL, focusing on sales performance, profitability, customer behavior, shipping efficiency, and operational performance. Finally, the transformed data was imported into **Power BI** to build interactive dashboards that present key insights and support data-driven decision-making.

---

## Project Workflow

### 1. Data Cleaning (Python)

* Removed irrelevant and redundant columns.
* Performed data cleaning and validation.
* Prepared the dataset for data warehousing and analysis.

### 2. Database Design (MySQL)

* Created the `supply_chain` database.
* Designed and implemented a **Star Schema**.
* Created dimension and fact tables.
* Loaded cleaned data into the data warehouse while maintaining referential integrity.

* Solved **20 business case study questions** using SQL.
