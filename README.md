Supply_chain_business_case_study_project
Overview
This project demonstrates my SQL problem-solving skills through the analysis of supply chain data. The project involves cleaning data using python,removing unnecessary columns and make it ready for analysis.Imported cleaned data from python to sql workbench for Star-schema design ,complex SQL business queries, and generatiing Insights and Actionable recommendation.

Project Structure
Data Cleaning: Handling invalid columns and ensuring data integrity.
Database Setup: Creation of the supply_chain database and perform star schema data modeling
Data Import: Import fact and dimension tables in Power BI desktop.
Business Problems: Solving 20 specific business problems using SQL queries.
Visualizations: Visualize insights through reporting
